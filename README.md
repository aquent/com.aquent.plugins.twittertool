Twitter Viewtool
=================================================
A plugin for dotCMS that adds a viewtool for interacting with Twitter

Installation
==================================
* Download and extract the plugin package to your {DOTCMS_ROOT}/plugins directory
* You will need to get twitter auth tokens from twitter
* Configure Twitter4J
